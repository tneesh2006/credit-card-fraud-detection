# Credit Card Fraud Detection

Machine learning project to detect fraudulent credit card transactions.

## Objective
To identify fraudulent credit card transactions using machine learning models.

## Dataset
Credit Card Fraud Detection Dataset (Kaggle)  
https://www.kaggle.com/datasets/mlg-ulb/creditcardfraud

Note: The dataset is not included in this repository due to file size constraints.

## Tools & Technologies
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Seaborn
- Plotly

## Environment
Google Colab
